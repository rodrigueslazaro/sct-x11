# Set Color Temperature

Set color temperature and brightness in X11.

Run: sct TEMPERATURE BRIGHTNESS

Where TEMPERATURE: 1000-10000 Kelvin and BRIGHTNESS: 0.1-1.0 %

Install:
- make
- sudo make install

Uninstall: 
- remove sct file from /usr/bin/
